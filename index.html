Kostplan — referanse
Refluks-tilpassa · Ingen hardkokte egg · Ingen fisk som hovudrett · 2300–2600 kcal/dag · Protein 180–215g (rå) · Karbo 255–310g (kokt)
Faste reglar: Fredag = taco · Laurdag = biff · Søndag = tandori (odde) / kjøttkaker (partalsveker)
Middagsrotasjon — veke A/B/C
Veke — denne veka. Rotasjonen baserast på vekenummer: veke % 3 = 0 → A · 1 → B · 2 → C.
